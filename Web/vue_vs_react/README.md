## vue-todo

[vue-todo](https://github.com/sunil-sandhu/vue-todo)

먼저 vue와 vue-cli를 글로벌로 설치한다.

```
npm install -g vue @vue/cli
```

이후 `vue-cli`를 이용하여 vue-todo 프로젝트를 생성한다.
이 때, `babel`, `eslint`를 사용하는 default로 생성하자.

```
vue create vue-todo
```
